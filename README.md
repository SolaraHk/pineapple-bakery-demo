# Pineapple Bakery 鳳梨餅家 — Website Demo

A polished Vite + React website demo for **Pineapple Bakery 鳳梨餅家** in Sheung Wan, Hong Kong.

This is a speculative sales/demo artifact only. It is **not affiliated with Pineapple Bakery** and should not be presented as official without client approval.

## Current website

The current production website is the final design that was previously developed as “v2”. The old v1 implementation has been removed from the source so the repo now has one active website only.

Public site:

```text
https://sloppy0726.github.io/pineapple-bakery-demo/
```

Public routes generated by the build:

- `/`
- `/menu/`
- `/schedule/`
- `/about/`
- `/faq/`

## What was built

- Mobile-first Vite + React website.
- Premium warm bakery visual direction: pineapple/golden tones, cream background, real public Instagram/social thumbnails for draft imagery, strong hero section.
- Dedicated pages for menu, schedule, about, and FAQ.
- Clickable menu product cards with product-detail popups including introduction and ingredients.
- English / Traditional Chinese language toggle.
- Instagram-first CTA flow.
- Footer with visible demo disclaimer.
- Outreach copy in English and Traditional Chinese in `OUTREACH.md`.

## Run locally

```bash
npm install
npm run dev
```

Then open the local Vite URL, usually:

```text
http://127.0.0.1:5173/
```

Production build check:

```bash
npm run build
```

Local verification after starting a preview server:

```bash
npm run verify -- http://127.0.0.1:4173/pineapple-bakery-demo/
```
